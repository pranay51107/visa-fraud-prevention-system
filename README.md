# Visa Fraud Prevention System

## Problem Statement
Every year, millions of visa applicants fall victim to fraudulent consultants and forged documents, leading to financial loss and visa rejections.

## Solution
An AI-powered platform that verifies visa consultants, checks document authenticity, predicts success likelihood, and alerts users about fraud risks before application submission.

## Key Features
1. Visa Consultant Verification Engine
2. Document Authenticity Checker
3. Crowdsourced Success Rate Database
4. Official Document Success Generator
5. Red Flag Alert System

## Tech Stack

- Google Natural Language API
- Google Custom Search API
- Firebase
- Google Maps API
- Google Translate API
